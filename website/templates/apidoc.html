API Routes

  • /api/v1/ - Returns endpoints.
  • /api/v1/<string:state>/ - Returns population flow to every other state from given state for all years.
  • /api/v1/<int:year>/ - Returns population flow between every pair of states for given years.
  • /api/v1/<string:state>/<int:year>/ - Returns population flow to every other state from given state for given year.

Example usage

Calling the endpoint /api/v1/florida/2017/ returns a JSON object:
{
 2017: {
   Alabama: 14171,
   Alaska: 906,
   Arizona: 8522,
   Arkansas: 1525,
   California: 24611,
   Colorado: 12385,
   Connecticut: 6589,
   ...
   }
}.

The interpretation of this is: in the year 2017, a total of 14171 people moved from Florida to Alabama, 906 people moved from Florida to Alaska, etc.

Calling the endpoint /api/v1/2017/ returns the JSON object:
{
  Alabama: {
   Alaska: 942,
   Arizona: 1337,
   Arkansas: 574,
   California: 3715,
   Colorado: 3120,
   ...
   }
}.

The interpretation of this is: in the year 2017, a total of 942 people moved from Alabama to Alaska, etc.
